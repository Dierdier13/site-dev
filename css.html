Initiation au CSS

Texte Explicatif : Qu'est-ce que le CSS ?


Le CSS (Cascading Style Sheets), ou feuilles de style en cascade, est un langage utilisé pour décrire la présentation visuelle des documents HTML. Alors que le HTML est responsable de la structure du contenu, le CSS détermine l'apparence de ce contenu : couleurs, polices, mise en page, tailles, marges, etc. Grâce au CSS, vous pouvez transformer une simple page HTML en une interface esthétiquement agréable et facile à naviguer.
Le CSS fonctionne en associant des styles à des éléments HTML spécifiques, en définissant comment ces éléments doivent apparaître lorsqu'ils sont rendus par un navigateur. Chaque règle CSS est composée d'un sélecteur (l'élément auquel le style sera appliqué) et de propriétés avec leurs valeurs (ce qui détermine l'apparence).

Exemple de base en CSS


Voici un exemple simple de fichier CSS qui modifie l'apparence d'une page HTML :

  body {
    background-color: #f0f0f0;
    font-family: Arial, sans-serif;
  }

  h1 {
    color: #333;
    text-align: center;
  }

  p {
    font-size: 16px;
    color: #666;
    line-height: 1.5;
  }

  a {
    color: #3498db;
    text-decoration: non
  }


  a:hover {
    color: #2980b9;
    text-decoration: underline;
  }

Explication du code :

body : Sélectionne l'élément <body> et lui applique un fond gris clair et la police Arial.
h1 : Modifie les titres <h1> en les rendant de couleur gris foncé et les centre sur la page.
p : Stylise les paragraphes <p> en définissant une taille de texte de 16px, une couleur gris clair et un espacement vertical agréable entre les lignes.
a : Définit la couleur des liens <a> en bleu et supprime le soulignement par défaut.
a:hover : Applique un style supplémentaire lors du survol d'un lien, en changeant sa couleur et en rétablissant le soulignement.




Les Sélecteurs CSS les plus courants


Les sélecteurs CSS permettent de cibler les éléments HTML auxquels vous voulez appliquer des styles. Voici quelques sélecteurs couramment utilisés :

1. Les Sélecteurs d'Éléments

Les sélecteurs d'éléments ciblent directement les balises HTML (comme <p>, <h1>, <div>, etc.)

  p {
    color: #333;
  }


2. Les Sélecteurs de Classe (.)

Les classes permettent de styliser des éléments de manière spécifique et peuvent être appliquées à plusieurs éléments sur une page. Les classes sont précédées d’un point (.) dans le CSS.

  HTML :

  <p class="important">Ceci est un paragraphe important.</p>


  CSS :

  .important {
    font-weight: bold;
    color: red;
  }


3. Les Sélecteurs d'ID (#)

Les identifiants (ID) sont uniques et ne peuvent être utilisés qu'une seule fois par page. Les ID sont précédés d’un dièse (#) dans le CSS.

  HTML :

  <h1 id="titre-principal">Bienvenue</h1>

  CSS :

  #titre-principal {
    font-size: 2em;
    color: #2c3e50;
  }


4. Les Sélecteurs de Groupe

Vous pouvez appliquer le même style à plusieurs éléments en les regroupant dans une seule règle, séparés par des virgules.

  h1, h2, h3 {
    font-family: 'Arial', sans-serif;
    color: #333;   }


Propriétés CSS couramment utilisées


Voici un aperçu des propriétés CSS les plus fréquemment utilisées pour styliser une page web :

1. Propriétés de Couleur

color : Définit la couleur du texte.

  p {
    color: #555;
  }

background-color : Définit la couleur de fond d'un élément.

  body {
    background-color: #f9f9f9;
  }


2. Propriétés de Texte

font-size : Détermine la taille du texte.

  h1 {
    font-size: 2.5em;
  }

font-family : Définit la police de caractères.

  body {
    font-family: 'Verdana', sans-serif;
  }

text-align : Aligne le texte horizontalement (par exemple, centré, à gauche, à droite).

  h1 {
    text-align: center;
  }


3. Propriétés de Boîte et de Marges

padding : Définit l'espace intérieur entre le contenu et les bords d'un élément.

  div {
    padding: 20px;
  }

margin : Définit l'espace extérieur entre un élément et ses voisins.

  div {
    margin: 10px;
  }

border : Ajoute une bordure autour d'un élément.

  p {
    border: 1px solid #ccc;
  }


4. Propriétés de Mise en Page

width et height : Définissent la largeur et la hauteur d'un élément.

  div {
    width: 300px;
    height: 200px;
  }

display : Définit le type d'affichage d'un élément (par exemple, block, inline, flex).

  a {
    display: block;
  }

position : Gère la position d'un élément sur la page (par exemple, relative, absolute, fixed).

  div {
    position: absolute;
    top: 50px;
    left: 100px;
  }

Les Méthodes pour Appliquer le CSS


Il existe plusieurs façons d'intégrer le CSS dans une page HTML :

1. CSS Interne

Le CSS interne est placé directement dans le fichier HTML, à l'intérieur de la balise <style> dans la section <head>.

  <head>
    <style>
  p {
    color: blue;
  }
    </style>
  </head>


2. CSS Externe

La méthode la plus courante consiste à lier un fichier CSS externe via la balise <link>.

HTML :

  <head>
    <link rel="stylesheet" href="style.css">
  </head>

Fichier style.css :

  p {
    color: blue;
  }


3. CSS En Ligne

Le CSS en ligne est appliqué directement à un élément HTML via l'attribut style. Cependant, cette méthode est déconseillée car elle mélange structure et présentation.

HTML :

  <p style="color: blue;">Ceci est un paragraphe en bleu.</p>

Conclusion


Le CSS est un outil puissant qui permet de séparer la présentation d'une page web de sa structure HTML. En maîtrisant les bases des sélecteurs, des propriétés et des méthodes d'intégration, vous pourrez facilement personnaliser l'apparence de vos pages web pour offrir une meilleure expérience utilisateur. Associé au HTML, le CSS forme la base de tout site web moderne, et une compréhension approfondie de ces deux langages est essentielle pour tout développeur web.
