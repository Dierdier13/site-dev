Initiation au HTML

Texte Explicatif : Qu'est-ce que le HTML ?


Le HTML (HyperText Markup Language) est le langage de base utilisé pour structurer le contenu des pages web. Il permet de définir la mise en page d'un site web en organisant les éléments de manière hiérarchique. Avec le HTML, on ne programme pas à proprement parler, mais on crée une structure logique qui sera interprétée par le navigateur pour afficher du texte, des images, des liens, des tableaux, des formulaires, et bien plus encore. HTML est constitué d'une série d'éléments appelés balises. Ces balises entourent le contenu que vous voulez afficher et indiquent au navigateur comment ce contenu doit être traité. Chaque balise HTML a une ouverture (par exemple <p>) et une fermeture (par exemple </p>), bien que certaines balises puissent être auto-fermantes (par exemple <img /> pour insérer une image).

Exemple de base en HTML


Voici un exemple simple de code HTML qui représente une structure typique d'une page web:

<!DOCTYPE html>
<html lang="fr">
<head>
  <meta charset="UTF-8">
  <title>Ma première page HTML</title>
</head>
<body>
  <h1>Bienvenue sur ma page</h1>
  <p>Ceci est un paragraphe de texte qui explique le contenu de la page.</p>
  <a href="https://www.example.com">Cliquez ici pour visiter un autre site</a>
</body>
</html

Explication du code :

<!DOCTYPE html> : Indique au navigateur que le document est écrit en HTML5, la version actuelle de HTML.
<html> : Cette balise contient tout le contenu de la page web.
<head> : C'est ici que sont définies les métadonnées, telles que le titre de la page (<title>) ou l'encodage des caractères
(<meta charset="UTF-8">).
<body> : Cette section contient tout le contenu visible de la page, comme le texte, les images, les liens, etc. <h1> : Utilisé pour les titres principaux (niveau 1).
<p> : Balise de paragraphe, utilisée pour afficher du texte.
<a> : Balise de lien hypertexte, qui permet de créer un lien vers une autre page ou un autre site.




Les balises HTML les plus courantes


Voici quelques balises couramment utilisées en HTML pour créer du contenu :

1. Les Titres (<h1> à <h6>)

Les titres sont utilisés pour structurer le contenu de la page. Ils vont du plus important (<h1>) au moins important (<h6>).

  <h1>Titre principal</h1>
  <h2>Sous-titre</h2>
  <h3>Sous-sous-titre</h3>


2. Les Paragraphes (<p>)

Les balises <p> sont utilisées pour délimiter des blocs de texte, appelés paragraphes.

  <p>Ceci est un paragraphe de texte simple.</p>


3. Les Liens (<a>)

La balise <a> est utilisée pour créer des hyperliens. L'attribut href définit l'URL vers laquelle le lien dirige.

  <a href="https://www.example.com">Visitez notre site</a>


4. Les Images (<img>)

La balise <img> permet d'insérer des images. L'attribut src indique le chemin de l'image, et l'attribut alt fournit une description alternative en cas de problème d'affichage.

  <img src="image.jpg" alt="Description de l'image" />


5. Les Listes (<ul>,<ol>,<li>)

Les listes permettent de structurer des éléments sous forme de liste à puces (<ul>) ou numérotée (<ol>). Chaque élément de la liste est contenu dans une balise <li>.

  <ul>
    <li>Premier élément</li>
    <li>Deuxième élément</li>
  </ul>

   <ol>
    <li>Premier élément</li>
    <li>Deuxième élément</li>
  </ol>



6. Les Tableaux (<table>)

Les tableaux permettent d'organiser des données sous forme de colonnes et de lignes. Voici un exemple simple :

  <table>
    <tr>
      <th>Nom</th>
      <th>Âge</th>
    </tr>
    <tr>
      <td>Jean</td>
      <td>25</td>
    </tr>
    <tr>
      <td>Marie</td>
      <td>30</td>
    </tr>
  </table>


7. Les Formulaires (<form>)

Les formulaires permettent aux utilisateurs d'envoyer des données à un serveur. Ils contiennent divers champs de saisie comme des cases à cocher, des boutons radio, des zones de texte, etc.

  <form action="/envoyer" method="post">
    <label for="nom">Nom :</label>
    <input type="text" id="nom" name="nom" />

    <label for="email">Email :</label>
    <input type="email" id="email" name="email" />

    <input type="submit" value="Envoyer" />
  </form>

Conclusion


Le HTML est un langage fondamental pour tout site web. Il ne s'agit pas d'un langage de programmation, mais d'un langage de balisage qui structure les informations. Comprendre et maîtriser les balises de base du HTML est essentiel pour créer des pages web fonctionnelles et bien structurées. En combinant le HTML avec d'autres technologies comme CSS (pour le style) et JavaScript (pour l'interactivité), vous pourrez concevoir des sites web complets et interactifs.
